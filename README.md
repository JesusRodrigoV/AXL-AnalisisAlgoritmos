# Proyecto Algoritmos

Para iniciar el proyecto primero en consola:

```bash
bun install
```

Si Bun no está instalado, instalarlo con:

Para Linux o mac:

```bash
curl -fsSL https://bun.sh/install | bash
```

Para Windows:

```Bash
npm install -g bun
```

ó

```PowerShell
powershell -c "irm bun.sh/install.ps1|iex"
```

Ejecutar

```bash
bun install
```

Iniciar :

```bash
ng serve
```
